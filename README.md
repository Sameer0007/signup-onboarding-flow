# Multi-Step Onboarding Flow

A modern, responsive multi-step registration interface built with React and TypeScript. This project features real-time validation, dynamic country code selection, and a secure password creation flow.

## 🚀 Features
- **Dynamic Account Selection**: Choose between Personal or Business accounts.
- **Phone Verification**: Custom country code dropdown with flag integration.
- **Smart OTP Input**: 4-digit numeric entry with auto-focus shifting.
- **Real-time Password Validation**: Live checklist for security requirements.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **State Management**: Built using React Context API for clean data handling.

## 🛠️ Tech Stack
- **React** (Vite)
- **TypeScript**
- **CSS Modules** (Scoped styling)
- **Lucide React** (Icons)
